# Flum ロードマップ

## フェーズ1: 基本機能ブラッシュアップ ← 現在地（〜2026年3月）

無料で使える基本機能を磨いてプロダクトの価値を確立する。

- フィード体験の完成（エラー表示、キーボード操作、スワイプリーダー、コマンドパレット）
- 技術基盤（middleware、PWA）

**ゴール**: 「毎日使いたい」と思えるフィードリーダーになっている

## フェーズ2: リリース準備（2026年3〜4月）

公開に必要なインフラと導線を整える。

- デプロイ（Vercel + 独自ドメイン）
- LP（ビジュアル、コピー、オタク層に刺さる見せ方）
- オンボーディング改善（初回体験で離脱させない）
- 認証追加（Google、Discord — GitHub だけだとハードルが高い）
- 利用規約・プライバシーポリシー
- アナリティクス導入（ユーザー数・利用状況の計測）
- **プロモーション開始** — 開発過程を X / Zenn で発信。「作ってる段階」から認知を取る
  - 開発ログ記事（設計判断、ハマりポイント、技術選定の理由）
  - スクショ・GIF で見た目のインパクトを先出し
  - ポートフォリオ記事も兼ねる

**ゴール**: 誰でもアクセスしてすぐ使い始められる状態。リリース前から認知がある
**目標**: 2026年4月リリース

## フェーズ3: ローンチ + 初期ユーザー獲得（2026年4〜5月）

公開して反応を見る。

- **ローンチ告知** — X / Zenn / はてブで記事投稿、デモ動画
- フィードバック導線（問い合わせフォーム or GitHub Issues）
- バグ対応・改善サイクル
- プリセットの拡充（オタク向けフィード: アニメ、ゲーム、VTuber 等）
- **継続発信** — ユーザーの声をリポスト、アップデート告知、使い方 Tips

**ゴール**: 実際のユーザーからフィードバックを得ている

## フェーズ4: マネタイズ（2026年夏〜）

ユーザーがついた段階で収益化を開始。

### 収益源
- 広告（AdSense — 補助的）
- アフィリエイト（チャンネルジャンルに合わせた商品リンク）
- フリーミアム（Pro プラン）

### 有料機能の候補

| 機能 | 説明 | 優先度 |
|------|------|--------|
| チャンネル数上限解放 | 無料は3〜5ch、Pro で無制限 | 高 |
| AI 要約・分類 | 記事の自動要約、チャンネルへの自動振り分け | 高 |
| 広告非表示 | フリーミアムの定番 | 高 |
| 壁紙拡張 | 無料は既定壁紙のみ、Pro で自由アップロード | 中 |
| ソース数上限解放 | チャンネルあたりのソース数制限を解除 | 中 |
| フィード保持期間延長 | 無料は24h表示、Pro で7日 | 中 |

**ゴール**: 月の収益が発生している
